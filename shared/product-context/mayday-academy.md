# Mayday Academy — Module Context

**Baseline** : Transformez la connaissance en compétence. **Key metric** : −48% onboarding time.

Integrated training platform built on top of the knowledge base. Converts business knowledge into skills through gamified learning. Content is authored in the same editor as Mayday Knowledge.

## Core Objects
- **Module** — core learning unit (standalone or part of a path); contains lesson sheets and quizzes; supports SCORM
- **Learning Path (Parcours)** — ordered sequence of modules visualized as a gamified map
- **Lesson Sheet** — content unit within a module (text, video, PDF, Google Slides, GIF, images); can embed Mayday Knowledge content directly
- **Quiz** — assessments within modules; AI-generated via Quiz Generator agent

## Key Features
- **Content management** — modules and paths have Draft → Published status; version control with archive access; content labels for organization and search filtering
- **Gamification (Dojo Universe)** — avatar, points, badges, levels; martial arts progression metaphor (novice → Sensei); leaderboards and challenges planned
- **Blended Learning** — synchronous workshops (ateliers) with scheduling and participant invitations; face-to-face, remote, and hybrid
- **Assignment** — manual or automatic (by group, role, label, or user attributes)
- **Notifications** — admins send notifications to learners (e.g. new module assigned); configurable per user
- **Search** — searchable from both Mayday Admin (KM view) and Mayday Desk (learner view)
- **Analytics** — module completion, time-spent, average scores, SCORM tracking, data export
- **Feedback loop** — agents send feedback on modules; surfaced to admins

## Interfaces
- **Mayday Admin** — KM creates and manages modules, paths, and ateliers
- **Mayday Desk** — agents access and complete their assigned training

## Languages Supported
22 languages including FR, EN, DE, ES, IT, PT, NL, PL, JA and more.
